UNIX

Ken Thompson & Dennis Ritchie
Criaram o Unix em 1969~1973 no Bell Labs (AT&T)
Brincando com um Raspberry PDP-11

O que esse UNIX tinha de tão legal?

  • Multi-tarefa e multi-usuário
    •
  • Uso de arquivos como abstração de dispositivos e outros objetos
    • Tudo é arquivo
    • Pipe (Streams)
    •
  • Escrito em C
    • Linguagem criada por Dennis Ritchie
    • Alto nível
    • Tornou o sistema portável
    •

#!/usr/bin/python2
import gtk

class PyApp(gtk.Window):
    def __init__(self):
        super(PyApp, self).__init__()

        self.set_title("Buttons")
        self.set_size_request(250, 200)
        self.set_position(gtk.WIN_POS_CENTER)

        btn1 = gtk.Button("Click me")
        btn1.connect("clicked", self.on_clicked)

        fixed = gtk.Fixed()
        fixed.put(btn1, 20, 30)

        self.connect("destroy", gtk.main_quit)
        self.add(fixed)
        self.show_all()

    def on_clicked(self, widget):
        md = gtk.MessageDialog(self,
            gtk.DIALOG_DESTROY_WITH_PARENT, gtk.MESSAGE_INFO,
            gtk.BUTTONS_CLOSE, "Linux Rules!")
        md.run()
        md.destroy()

SSH

Secure Shell
Gerando uma par de chaves RSA 

[andre@mars ~]$  ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

Serão criados os arquivos id_rsa(privado) e id_rsa.pub(público) em ~/.ssh

Acessando um servidor 

[andre@mars ~]$ ssh root@192.168.1.10

A chave pública (id_rsa.pub) do usuário (andre) deve estar no arquivo /root/.ssh/authorized_keys

SCP (Secure Copy) 

# From local to remote
$ scp your_username@remotehost.edu:foobar.txt /some/local/directory

# From remote to local
$ scp foobar.txt your_username@remotehost.edu:/some/remote/directory
